# Agents Guide — Compourri Software Essentials

## Post-Upstream-Merge Workflow

This repo is a fork of ChrisTitusTech/winutil. After merging upstream changes:

1. Run the branding script to restore Compourri branding:
   ```powershell
   .\scripts\Apply-Branding.ps1
   ```
2. Review the diff with `git diff`
3. Commit the branding patches

All branding locations are documented in `BRANDING.md`.
